## Overview

This MQTT Client Application facilitates with all the MQTT related operations while using Eclipse Kura for IoT Application development.

The main motive of developing this application is to use Eclipse Kuraâ€™s CRUD like feature provided by CloudLet. This requires all the payloads to be encoded and decoded using Google Protocol Buffer. If we use any of the widely available MQTT Client Tools, we can only be able to use MQTT Payloads of simple string type. This utility application provides you with all the opportunities to encode and decode your provided payload on the fly.
